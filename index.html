Minecraft Web Inventory

Как создать такой инвентарь:

  1. Создайте HTML-элемент

                <div id="case-1" data-size="1"></div>

  2. Подготовьте объект со списком предметов

                itemsList: [
  {
    itemName: "Diamond_Pickaxe",
    displayName: "&fАлмазная кирка",
    description: ["\n", 
    "&9+5 к урону", 
    "Прочность: 131/131"]
  }
]

  3. Создайте объект класса MinecraftWebInventory

    Подайте в объект список предметов и объект конфигурации

                new MinecraftWebInventory(listOfItems, config)

  4. Вызовите метод createTable() созданного объекта

                MinecraftWebInventory.createTable(HTML-элемент)

Что делает параметр data-size?

Данный параметр указывает сколько ячеек инвентаря будут в одной строке. Данный параметр можно менять динамически.

Этот параметр, кстати, не обязателен.
